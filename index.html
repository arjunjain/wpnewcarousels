WPNewCarousel

Open Source Wordpress plugin to generate carousel

Download as .zip Download as .tar.gz View on GitHub

About

WPNewCarousels is a wordpress plugin which is used to create the carousel that can be inserted to wordpress page/post using shortcode. This plugin also support wordpress multisite setup.

How to use

Insert this short code in wordpress page/post

[wpnewcarousel name="CAROUSEL_NAME" height="" width="800" effect="" startslide="" animationspeed="" imagepausetime="" shownav="" hoverpause=""]

Integration with wordpress theme

If you want to integrate the wpnewcarousel with wordpress theme than you need to add following php code in theme php file (Ex: header.php)

echo do_shortcode('[wpnewcarousel name="CAROUSEL_NAME" height="" width="" effect="" startslide="" animationspeed="" imagepausetime="" shownav="" hoverpause=""]');
